# MiniProject 2: Language Model

This project implements a simple character-level GPT language model trained on Shakespeare text.

## Features

- PyTorch implementation of a mini GPT
- Character-level dataset and encoding
- Training and generation scripts
- WANDB integration for experiment tracking

## Usage

1. Install requirements:
    ```
    pip install requirements.txt
    ```

2. Run the notebook:
    - Open `my_model.ipynb` in VS Code or Jupyter.
    - Run all cells to run the sweep, train with the best config combo found and generate text.

3. Generate text:
    - Edit the `prompt` variable in the last cell to try your own prompt.
